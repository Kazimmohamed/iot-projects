This is a Node.js + Express-based dashboard for real-time temperature and humidity data from an ESP32 using DHT11 sensor.

## Features
- ESP32 in STA mode
- Data in JSON via `/data`
- Node.js Express server
- Frontend UI with HTML/CSS/JS
- Realtime updates via Fetch API

## Tech Stack
- ESP32 + DHT11
- Node.js + Express
- HTML5 + CSS3 + JavaScript
